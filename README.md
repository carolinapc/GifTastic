# GifTastic

A simple use of Giphy and Bands in Town API's.

On this website it is possible to store the favorites stickers and follow the preferred artists.

[Enjoy it!](https://carolinapc.github.io/GifTastic/)

## Technologies Used

- Javascript
- jQuery
- Ajax

##### API's used

- Giphy
- Bandsintown
